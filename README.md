# 🚀 Featured Projects

## 🧠 GitMind — AI Repository Intelligence Platform
### Analyze any public GitHub repository end-to-end with AI

**Live Demo:** https://gitmindai.vercel.app/  
**Repository:** https://github.com/ArunChandrasekar07/gitmind.git  

Full-stack AI SaaS that analyzes GitHub repositories, commit history, and code diffs to detect risk patterns and repository health.

### Key Features
- ⚡ Processes repositories in **< 30 seconds**
- 🔁 Multi-LLM failover: **Gemini → Groq → OpenRouter**
- 🧵 Multi-threaded commit analysis via `ThreadPoolExecutor`
- 🔐 OAuth + Supabase authentication
- 📊 Repository health score with AI-generated insights
- 🚦 Risk detection: **Safe / Warning / High Risk**

### Architecture
```text
GitHub API
   ↓
FastAPI Backend
   ↓
ThreadPool Executor
   ↓
LLM Router
   ↓
Risk Engine
   ↓
PostgreSQL + Supabase
   ↓
Next.js Dashboard
```

**Tech Stack**  
`Next.js` `TypeScript` `FastAPI` `PostgreSQL` `Supabase` `Gemini` `Docker`
